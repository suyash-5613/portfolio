# 🚀 Suyash Sahu | Software Developer Portfolio

Welcome to the repository for my personal developer portfolio! This project showcases my journey, my core skills, and some of my favorite recent works across full-stack development.

## ✨ Features

- **Pristine Dark Mode UI**: Seamlessly integrated global theme toggling with synchronized glowing neons (Electric Blue and Pink).
- **Glassmorphism Components**: A sleek, fully responsive design using frosted glass effects across the Navigation and Cards.
- **Bento Grid Architecture**: Clean, card-based information display designed for readability and premium aesthetics in the "About" section.
- **Interactive Project Showcase**: Smoothly animated project cards that trigger sliding overlay reveals and technology stack badges.

## 🛠️ Built With

- **React** (Bootstrapped with CRA)
- **TypeScript**
- **Vanilla CSS** (for precise, lightweight styling control and premium layout grids)
- **React-Icons**

## 🚀 Getting Started

To run this project locally, simply follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/suyash-5613/react-portfolio.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm start
   ```
   *The app will automatically open at [http://localhost:3000](http://localhost:3000).*
